# Gene Expression Prediction Modeling from Histone Modification using CNN and Attention Models

More information can be found at:
[Gene Expression Prediction Modeling](https://www.overleaf.com/read/zwvkqqnssqvx#241ca3)

## Overview
This project predicts gene expression states from histone modification signals using deep learning models.

## Technologies Used
- Python
- PyTorch
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Models Implemented
- CNN
- Attention-based BiLSTM

## Dataset
Histone modification and gene expression dataset.

## Evaluation Metrics
- Accuracy
- F1 Score
- AUC
